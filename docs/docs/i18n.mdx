# Internationalization
we use dayjs for date and time parsing and formatting.

dayjs has [excellent i18n support](https://day.js.org/docs/en/i18n/i18n)

we use the `locale` prop to set the locale for the component.
currently supported locales are:

if you want to extend it PRs are welcome, or drop an issue in our github

```tsx
<DatePicker locale="fr|en|de|it|ja|ko|nb|pl|pt|es|sv|tr" />
```

- ar - Arabic
- zh - Simplified Chinese
- da - Danish
- nl - Dutch
- en - English
- fr - French
- de - German
- it - Italian
- ja - Japanese
- ko - Korean
